# comandos branch
- git branch		-> lista todas as branchs
- git branch -b <nome>	-> cria uma nova branch [secundaria01, secundaria02...]
- git branch <nome>	-> muda para a branch específica
- git checkout <nome>	-> muda para a branch especificada
- mit merge <nome>	-> mescla a branch atual com a branch especificada
